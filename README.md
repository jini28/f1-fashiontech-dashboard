# From Grid to Wardrobe: F1 FashionTech Dashboard

## Overview

This project explores the relationship between Formula 1 team performance and fashion/merchandise search demand. Formula 1 has increasingly become a lifestyle and fashion space, with teams and drivers influencing consumer interest in merch, streetwear, and brand collaborations.

The project combines Formula 1 race performance data with fashion-related Google Trends data to analyze whether stronger team performance is associated with higher search interest for F1 fashion and merchandise.

## Research Question

Do higher-performing Formula 1 teams generate stronger fashion and merchandise search demand?

## Data Sources

- Formula 1 race data from Kaggle
- Google Trends search interest for F1 fashion and merchandise-related terms

## Methods

The project includes:

- Data cleaning and preprocessing
- Team-level performance aggregation
- Google Trends data collection
- Fashion demand score creation
- Performance vs. demand analysis
- Interactive dashboard development

## Dashboard Features

- KPI cards
- Fashion search interest over time
- Performance vs. fashion demand scatterplot
- Team fashion demand rankings
- Team performance rankings
- Key insights

## Tech Stack

- Python
- Pandas
- NumPy
- Plotly
- Streamlit
- Pytrends
- Scikit-learn
- Google Colab

## Key Insight

The analysis suggests that F1 fashion demand is not driven by performance alone. While race success can increase visibility, team legacy, driver identity, cultural relevance, and social media presence may also shape fashion and merchandise interest.
